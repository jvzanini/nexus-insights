interface BadgeOption<T> {

  label: string;
  description?: string;

### T4: AdvancedFilters component

**Files:**
- Criar `src/components/reports/advanced-filters.tsx` (`'use client'`)
- Criar `src/lib/reports/filter-state.ts` (serialize/deserialize/diff)

**API:**
```ts
interface AdvancedFiltersProps {
  inboxes: Array<{id: number; name: string}>;
  teams: Array<{id: number; name: string}>;
  assignees: Array<{id: number; name: string}>;
  initial: FilterState;
}
interface FilterState {
  period: PeriodKey;
  customRange?: { start: string; end: string };
  inboxIds: number[];
  teamIds: number[];
  assigneeIds: number[];
  statuses: number[];
  priorities: number[];
  search?: string;
}
```

**Comportamento:**
- Renderiza acima do conteúdo (não drawer).
- Linha 1: PeriodPills (Hoje/7d/30d/Personalizado).
- Linha 2: Multi-selects (Inbox, Departamento, Atendente, Status, Prioridade) + Search.
- Linha 3: Botões "Aplicar filtros" (primary) + "Limpar" (outline).
- Estado interno `draft` mutado pelos inputs.
- Botão "Aplicar" desabilitado se draft === applied (URL).
- Indicador "X filtros pendentes" quando há diff.
- Ao clicar Aplicar: `router.push(?...new URL params)`.

**filter-state.ts:**
- `serializeFilterState(state): URLSearchParams`
- `deserializeFilterState(params): FilterState`
- `diffStates(a, b): number` (count of differences)

**Query dashboard-data.ts:**
```ts
export interface DashboardData {
  stats: {
    received: number;          // recebidas no período
    resolved: number;          // resolvidas no período
    open: number;              // em aberto agora
    resolutionRate: number | null; // %
    comparison: { received: number | null; resolved: number | null; resolutionRate: number | null };
  };
  chart: Array<{bucket: Date; received: number; resolved: number}>;
  topAgents: Array<{name: string; avg_seconds: number}>;
  topInboxes: Array<{name: string; count: number}>;
  topTeams: Array<{name: string; count: number}>;
  recent: Array<{id: number; display_id: number; contact_name: string|null; inbox_name: string|null; assignee_name: string|null; status: number; last_activity_at: Date}>;
}

export async function dashboardData(args: {accountId: number; period: PeriodKey; tz: string}): Promise<DashboardData>
```

7 queries em Promise.all + cache 30s.

**StatsCard:** Recebidas/Resolvidas têm comparison (TrendingUp/Down + % vs período anterior). Em Aberto tem badge "agora". Taxa de resolução com %.

**ConversationsLineChart:** Recharts com 2 séries (Recebidas violet-500 / Resolvidas emerald-500). X axis: hora (today) ou dia (7d/30d).
